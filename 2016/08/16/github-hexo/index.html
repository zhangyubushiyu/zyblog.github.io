利用GitHub+Hexo搭建纯静态Blog网站（搭建篇）

发表于 2016-08-16   |   分类于 随手分享   |     |  

自从接触前端以来，心中有一个想法要搭建一个自己的独立博客。之前尝试wordpress，自己试用一段时间，功能虽然很强大，插件也很丰富，但是感觉不是自己想要的。后来刷知乎的时候发现hexo，不需要数据库也不需要服务器，而且使用起来简单主题美，最主要是湾湾同胞开发。
